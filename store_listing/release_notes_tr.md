# Deprem Takip - Play Store Release Notes

## Versiyon 1.0.0

### Özellikler:
- 📍 Türkiye genelindeki deprem verilerini anlık takip
- 🗺️ İnteraktif harita üzerinde deprem konumları
- 🔔 Anlık deprem bildirimleri
- 📊 Deprem büyüklüğü ve derinlik bilgileri
- 📱 Konum tabanlı yakındaki depremler
- 🔗 Detaylı deprem bilgilerine erişim
- 📞 Acil durum numaraları
- 💾 Çevrimdışı veri depolama

### Teknik Özellikler:
- Modern ve kullanıcı dostu arayüz
- GPS tabanlı konum servisleri
- Güvenli veri iletişimi
- Düşük pil tüketimi
- Android 5.0+ desteği

### Güvenlik:
- Kullanıcı verilerinin güvenli işlenmesi
- Sadece gerekli izinler
- Şeffaf veri kullanımı
